# violao-ccb

## Start

### Browser

1. On your application

   ```sh
   expo start:web
   ```

2. Press `W` on keyboard

### Android

1. Activate DevMode on yor Android

2. Install `Expo Go` on your Android

3. Plug your android by USB on PC

4. On your application

   ```sh
   expo start --tunnel
   ```

5. Press `A` on keyboard
